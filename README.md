# Projeto QA - Testes de API com Postman

## Sobre o projeto

Projeto desenvolvido para praticar testes de API utilizando Postman.

## Objetivo

Validar endpoints de login, cadastro e consulta de usuários, aplicando conceitos de Quality Assurance.

## Ferramentas utilizadas

- Postman
- GitHub

## Cenários testados

- Buscar todos usuários
- Buscar usuário por Id
- Buscar usuário inexistente
- Criar usuário
- Atualizar usuário
- Excluir usuário

## Estrutura do projeto

- collections
- test-cases
- bug-reports
- evidencias_test
- evidencia_bugs
